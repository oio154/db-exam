Na adres email (lub inny kontakt), podany w szkole, otrzymałeś czteroliterowy identyfikator egzaminacyjny. Identyfikator ten oznaczony jest w egzaminie jako XXXX. Zamiast XXXX podajesz zawsze swój identyfikator.
Rozwiązanie zadań egzaminacynych następuje w bazie danych, do której otrzymujesz dostęp na czas egzaminu.
Podłączenie do bazy danych.
Podłącz się do bazy danych używając następujących danych
Host: s.web.katowice.pl
User: XXXX
Password: XXXX
Zadanie 1.
Utwórz tabelę t1
id a b c d
int, primary key, auto increment int int int int
Zadanie 2.
Utwórz tabelę t2
id t1_id li
int, primary key, auto increment int int
Zadanie 3.
Utwórz relację pomiędzy utworzonymi tabelami. Kluczem obcym w tabeli t2 jest pole t1_id w relacji do t1.id. Na podstawie informacji o właściwościach poszczególnych pól w tabelach ustal jakiego typu będzie to relacja: 1:1, 1:n czy n:n
Zadanie 4.
Do tabeli t1 wprowadź 2 dowolne rekordy, do tabeli t2 wprowadź 5 dowolnych rekordów. Pamiętaj o utworzonych relacjach!
Zadanie 5.
Utwórz widok sr1 zawierający średnią wartość pól a,b,c z tabeli t1;
Zadanie 6.
Utwórz raport (w formie widoku o nazwie sr2) zawierający ilość rekordów t2 przypadających na każdy rekord t1.
